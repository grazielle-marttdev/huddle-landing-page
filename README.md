# Frontend Mentor | Huddle landing page with single introductory section

Uma Landing Page com seção introdutória única, desenvolvida com conceitos de Flexbox, focando na responsividade.

## Demonstração do Layout para Desktop

![Preview da página](./src/design/design-desktop.png)

## Tecnologias utilizadas

- HTML5
- CSS3

## O que aprendi:

- Organização de arquivos e estrutura do projeto
- Utilização de HTML semântico e Flexbox
- Responsividade com *media queries* 

## Dificuldades encontradas

Tive problemas com a responsividade na parte do ```main```. A imagem começava a ficar "espremida" depois de certo ponto diminuido a tela, mas ao definir uma altura e largura de 100% e corrigir o uso do flexbox, a imagem se adaptou corretamente.
